### 1.3 Prepare 命令推演

如果从零使用 Prepare 命令探索 datametrics API，能否收集到同等信息？

| 阶段 | 可收集 | 不可收集 | 原因 |
|------|--------|---------|------|
| Phase 1: API 探索 | 工具名、参数、返回格式 | 快速计算语法、period 语法 | 这些是 API 的高级语法，不在 OpenAPI spec 中 |
| Phase 2: 用户访谈 | 铁律、语义规则、常见错误 | 完整的 timeConstraint 速查表 | 需要大量实际使用经验积累 |
| Phase 3: 知识结构化 | 自动完成 | — | — |

**结论**: Prepare 命令能收集约 60-70% 的信息。快速计算语法、period 语法、timeConstraint 速查表、composition constraints 等需要 API 文档或资深用户输入补充。Prepare 命令的 Phase 2 用户访谈需要被访者具备深度使用经验。
